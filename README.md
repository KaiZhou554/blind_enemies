# Blind Enemies

English | [简体中文](README_zh.md)

**Blind Enemies** is a Minecraft Bedrock Edition addon that makes most hostile mobs friendly toward players.

Hostile mobs will no longer attack players, and even if players attack them first, they will usually not retaliate.  
However, most mob-to-mob combat behavior is preserved, so the world still feels alive and dynamic.

This addon is ideal for players who:
- enjoy exploration without constant combat
- want a more relaxed survival experience
- prefer interacting with mobs rather than fighting them

![Demo Image](docs/demo.webp)

## Features

- Most hostile mobs become friendly to players
- Mobs usually do not retaliate when attacked
- Mob vs mob combat behavior is largely preserved
- Works in survival worlds without major gameplay changes

## Design Decisions

Some mobs are intentionally **not modified**:

- **Ender Dragon** 
- **Wither** 

Additionally:

- **Creaking** is currently not modified due to a bug in the base game that prevents proper behavior changes.

## Use Cases

This addon is perfect for:

- peaceful exploration
- creative building in survival worlds
- players who dislike constant combat
- accessibility-friendly gameplay

## Installation

1. Download the addon package.
2. Import it into Minecraft Bedrock Edition.
3. Enable it in your world settings.
